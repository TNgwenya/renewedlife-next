# Renewed Life International – Next.js Website Rebuild

This is the active Next.js rebuild for renewedlifeint.com.

## What is included
- Modern homepage
- About page
- Ministries page
- Sermons page
- Giving page
- Contact / Plan Your Visit page
- Responsive layout and custom styling
- Shared content model for church information and page content
- Integrated logo and real worship photography
- Verified Facebook, Instagram, and YouTube links
- Working first-version visitor and contact forms

## Run locally
```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
