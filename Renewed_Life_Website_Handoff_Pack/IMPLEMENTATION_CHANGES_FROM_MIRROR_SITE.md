# Changes We Are Going to Implement from the Website We Are Mirroring

## Overall principle
We are borrowing **structure, clarity, and user journey principles** — not cloning branding or copying exact design elements.

## 1. Homepage transformation
### Current issue
The current homepage does not strongly prioritize first-time guests.

### New approach
The new homepage should include:
- a clear hero statement
- service time/location visibility
- primary CTA: **Plan Your Visit**
- secondary CTAs: **Watch Online**, **Give**, **Connect**
- a section introducing the church briefly
- a section for ministries / next steps
- latest sermon highlight
- pastor / church welcome section
- testimonials or community highlights
- footer with strong contact visibility

## 2. Dedicated Plan Your Visit page
This is one of the biggest changes.

### It should include
- exact service times
- address and map
- welcome statement
- what to expect on Sunday
- children/family information if applicable
- what to wear / atmosphere reassurance
- what happens when a visitor arrives
- WhatsApp/contact option
- a simple visitor interest form

## 3. Better navigation and page priorities
The navigation should become clearer and simpler.

### Recommended nav
- Home
- Plan Your Visit
- About
- Ministries
- Sermons
- Give
- Contact

Optional:
- Events

## 4. Stronger content hierarchy
Each page should have:
- one clear purpose
- one primary action
- easy-to-scan sections
- concise, pastoral, welcoming language

## 5. More modern UI system
We will implement:
- cleaner spacing
- larger and more intentional typography
- consistent buttons
- consistent card layouts
- better mobile responsiveness
- stronger visual contrast where needed
- modern section rhythm

## 6. Better media and digital ministry pathways
We will strengthen:
- sermon access
- YouTube integration
- livestream / online church pathways
- giving visibility
- contact channels

## 7. Real photography and trust signals
We should move away from a generic template feel by using:
- real church photography
- ministry moments
- pastoral images
- event/community visuals
- real testimonies where available

## 8. Footer and contact improvement
The footer should become a strong utility block with:
- church address
- phone
- email
- service times
- social links
- quick links
- giving link

## 9. Copy refinement
We will improve wording to make it:
- clearer
- warmer
- more polished
- more grammatically consistent
- more compelling to the unchurched and first-time visitor
